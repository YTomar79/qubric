# Contributing

Thanks for your interest in improving Qubric.

## Getting set up

1. Fork and clone the repository.
2. Open `Qubric.xcodeproj` in Xcode 15 or later.
3. Build and run the `Qubric` scheme on an iOS 17+ simulator.

## Before opening a pull request

- Run the test suite (`⌘U` in Xcode, or the `xcodebuild test` command in the README). All tests must pass.
- Keep changes focused; one logical change per pull request.
- Match the existing code style and naming conventions.
- Update documentation when behavior changes.

CI runs the build and test suite on every pull request — please make sure it is green.

## Reporting issues

Open an issue with steps to reproduce, expected behavior, and the iOS version and device (or simulator) you observed it on.
