# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.1] - 2026-06-25

### Added

- Unit test target (`QubricTests`) covering the quantum simulation engine.
- Continuous integration workflow that builds the app and runs tests on every push and pull request.

### Changed

- Backend API URL is now supplied via configuration rather than a hardcoded value.
